# 📊 Painel Quinze | Dashboard de Gestão para Agências

Um sistema completo de **Business Intelligence (BI)** construído do zero para modernizar a gestão de agências de comunicação. O painel transforma dados brutos em visualizações dinâmicas, permitindo tomadas de decisão rápidas e baseadas em dados.

---

## ✨ Destaques do Projeto

Diferente de dashboards estáticos, este projeto utiliza **Google Sheets como Banco de Dados**. Através do Google Apps Script (API), o painel consome os dados da planilha em tempo real:
- ✅ **Sem custo de servidor de banco de dados.**
- ✅ **Atualização instantânea:** Editou na planilha, o gráfico muda no Dashboard.
- ✅ **Interface Intuitiva:** Foco total na experiência do usuário (UX).

## 🚀 Módulos (Dashboards)

O sistema conta com 7 visões estratégicas:

* 💰 **Financeiro:** Controle de faturamento, rentabilidade e evolução de jobs.
* 📈 **Produção Diária:** Monitoramento de esforço da equipe e entregas.
* 🎯 **NPS (Satisfação):** Gráfico de radar por setor e ranking de clientes.
* ⏱️ **Tempo & SLA:** Controle de prazos e identificação de gargalos.
* 👥 **Squads:** Visão de produtividade detalhada por time.
* 💼 **Carteiras:** Distribuição de clientes por responsável (Market Share).
* ⚙️ **Processos:** Acompanhamento de rotinas com barras de progresso animadas.

## 🛠️ Tecnologias Utilizadas

* **Frontend:** React, TypeScript, Vite
* **Estilização:** Tailwind CSS
* **Gráficos:** Recharts
* **Ícones:** Lucide React
* **Integração:** REST API consumindo Google Apps Script (Google Sheets)
